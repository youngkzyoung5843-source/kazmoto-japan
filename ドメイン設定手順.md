# kazmoto.jp ドメイン取得・DNS設定手順

**対象サイト**: https://github.com/youngkzyoung5843-source/kazmoto-japan  
**公開予定URL**: https://kazmoto.jp

WHOIS確認済み（2026-07-08）: `kazmoto.jp` は **未取得（NXDOMAIN）** — DNS が存在しないためカスタムドメインではアクセス不可。  
**暫定URL**: https://youngkzyoung5843-source.github.io/kazmoto-japan/

> `CNAME` ファイルは DNS 設定完了までリポジトリから外しています（外している間は github.io URL で公開）。

## Step 1 — ドメイン取得（オーナー作業）

以下いずれかで `kazmoto.jp` を取得してください。

| サービス | 目安料金 | 備考 |
|---------|---------|------|
| [お名前.com](https://www.onamae.com/) | 約 ¥128/年〜 | .jp取得実績が多い |
| [ムームードメイン](https://muumuu-domain.com/) | 約 ¥128/年〜 | 管理画面がシンプル |
| [Cloudflare Registrar](https://www.cloudflare.com/products/registrar/) | 原価相当 | DNSもCloudflareで一元管理可 |

**登録者情報**: Kazmoto合同会社（法人名義推奨）  
**取得後**: ネームサーバーはStep 2で設定

## Step 2 — DNS設定

ドメイン管理画面で以下を設定してください。

### ルートドメイン（kazmoto.jp）

| タイプ | ホスト | 値 |
|--------|--------|-----|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| AAAA | @ | 2606:50c0:2300::153 |
| AAAA | @ | 2606:50c0:2301::153 |
| AAAA | @ | 2606:50c0:2302::153 |
| AAAA | @ | 2606:50c0:2303::153 |

### www（任意・推奨）

| タイプ | ホスト | 値 |
|--------|--------|-----|
| CNAME | www | youngkzyoung5843-source.github.io |

---

## Step 3 — GitHub Pages 側（runa設定済み）

- リポジトリに `CNAME` ファイル（`kazmoto.jp`）を追加済み
- GitHub Pages の Custom domain を `kazmoto.jp` に設定済み

DNS反映後（数分〜48時間）、GitHub リポジトリの **Settings → Pages** で  
「DNS check successful」と表示されれば完了です。

---

## Step 4 — 確認

```bash
curl -I https://kazmoto.jp
```

ブラウザで https://kazmoto.jp が Kazmoto Japan トップを表示すればOK。
